# FLONK Screenshots

Place your FLONK game screenshots in this directory with the following naming:

- `flonk-1.png` - First screenshot
- `flonk-2.png` - Second screenshot
- `flonk-3.png` - Third screenshot
- `flonk-4.png` - Fourth screenshot

Recommended dimensions: 1170x2532 (iPhone aspect ratio 9:16)

The landing page will automatically display these screenshots in the gallery section.
